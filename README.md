# Scanbot PCF Barcode Scanner

This project is a **PowerApps Component Framework (PCF) control** that integrates the **Scanbot Web SDK** to provide a barcode scanning functionality in Power Apps. It uses **React** for the UI and demonstrates how to include Scanbot SDK's WebAssembly files in a PCF control.

---

## Features

- Custom PCF control for barcode scanning.
- React-based UI "Open Scanner" button.
- Supports Scanbot Web SDK initialization with WASM/JS engine.
- Easy integration with Power Apps.

---

## Prerequisites

- Node.js >= 16  
- npm >= 8  
- PowerApps CLI (pac) installed  
- React 18+  

---

## Installation

1. **Clone the repository:**

```
[Git Repo](https://github.com/ranwala/ScanbotSDK-PCF-Example.git)
```

2. **Install dependencies:**

```
cd scanbot-sdk-powerapps-example
npm install
npm run build
npm start
```
